# Python Cheatsheets

Python Cheatsheets is a repository created to offer quick and useful reference materials for typical Python tasks, particularly for workflows involving data analysis and machine learning. In addition to basic data preprocessing methods like feature scaling, exploratory data analysis (EDA), and data encoding techniques, it includes brief cheatsheets for popular libraries and concepts like NumPy, Pandas, and Matplotlib.

## 📦 Technologies

- `python`
- `numpy`
- `matplotlib`
- `seaborn`
- `scikit-learn`

## 👨‍🎓 The Process

This training program was designed to provide hands-on experience with Python for data analysis and machine learning:

1. **Foundation**: Started with Python basics—syntax, data types, and control flow.
2. **Data Manipulation**: Learned NumPy for numerical computing and Pandas for data wrangling.
3. **Visualization**: Explored Matplotlib and Seaborn to create meaningful visualizations.
4. **Machine Learning**: Applied scikit-learn algorithms to real-world datasets.
5. **Practice**: Completed exercises and projects to reinforce each concept.

## 🪧 Running the Project

To run the project in your local environment, follow these steps:

1. Clone the repository to your local machine.
2. Run `pip install -r requirements.txt` to install the required dependencies.
3. Run `start.bat` to open jupyter lab on your browser.
